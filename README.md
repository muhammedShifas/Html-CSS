# HTML Basic Tags 🌐

Welcome to my **HTML Basic Tags** repository!

## 📌 About

HTML (HyperText Markup Language) is the standard language used to create and structure web pages.  
This repository is created to practice and understand the fundamental HTML elements.

- Headings

- Links

- Lists
- Tables
- Forms
- Buttons
- Div and Span
- Semantic Tags
